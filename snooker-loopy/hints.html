Snooker Loopy - Hints

1:
Each image represents a two-word phrase where the first word is a colour.
2:
Each image's second word can be found on one of the tables.
3:
This isn't a normal wordsearch. Think about how a snooker ball might travel round the table.
4:
Use each word's associated colour to find the total score on each table, and convert each score to a letter.
